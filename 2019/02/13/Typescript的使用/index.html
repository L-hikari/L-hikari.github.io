Typescript的使用

发表于 2019-02-13 | 分类于 前段技能 | | 阅读次数: ℃

typescript

首先要安装 typescript

命令行运行 npm install -g typescript

编译

在项目目录下运行 tsc ts文件名 就可以将 ts 文件编译成 js文件

在 vscode 中配置自动编译

vscode 中进行配置可以自动将 ts 编译为 js文件，就不用每次都自己手动编译啦

  1. 进入项目目录下执行命令 tsc --init 生成一个 tsconfig.json 配置文件

  2. 在 tsconfig.json 文件中找到 outDir 选项将注释取消。这里写上你想输出的文件

  3. 然后在 vscode 中最上面菜单选项那里选择 任务 => 运行任务 => 点击 监视 tsconfig.json

    如果是英文版的话 点击 Terminal => Run Task => watch -tsconfig.json

类的多态性

多态: 父类定义一个方法不实现，让继承它的子类实现，每一个子类有不同的表现。多态属于继承

类的抽象
